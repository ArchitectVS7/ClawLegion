# Task 3: Frontend Web Page (Task Manager PWA)

**Complexity:** High  
**Time Limit:** 10 minutes  
**Type:** Full application development

---

## Requirements

Build a complete **Task Manager** progressive web app from scratch.

### Core Features

1. **Task List Display**
   - Show all tasks in a clean, organized list
   - Each task shows: title, status (pending/complete), timestamp

2. **Add New Task**
   - Input field + "Add Task" button
   - Tasks default to "pending" status

3. **Task Actions**
   - Mark as complete/incomplete (toggle)
   - Delete task
   - Edit task title (inline or modal)

4. **Filtering**
   - "All" / "Pending" / "Completed" filter buttons
   - Update view based on selected filter

5. **Persistence**
   - Save tasks to `localStorage`
   - Load tasks on page refresh

### Technical Requirements

- **No framework required** (vanilla JS acceptable, React/Vue optional)
- **Responsive design** (works on mobile + desktop)
- **Clean UI** (basic CSS, readable, organized)
- **Single HTML file** (or clear file structure if multi-file)

### UI Specifications

- **Color scheme:** Light background, blue accent color
- **Layout:** Header with title, input section, filter buttons, task list
- **Task item:** Checkbox, title text, delete button
