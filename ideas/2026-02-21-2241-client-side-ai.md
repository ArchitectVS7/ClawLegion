## Discovery

GitNexus - Zero-server code intelligence engine that runs entirely in browser. Creates knowledge graphs from GitHub repos with built-in Graph RAG agent. No backend required.

## First Principles Deconstruction

**Conventional assumption:** AI tooling needs servers, APIs, cloud compute

**Core question:** Where does the actual work happen in code intelligence?

**Answer:** In the browser, where the developer already is

**Follow-up:** Does knowledge graph need to exist anywhere else?

**Answer:** No - it's ephemeral context for exploration

## Selected: Infrastructure Inversion

**Why:** Highest combined score across all dimensions. Captures the fundamental architectural shift while being immediately actionable.

**The insight:** We've been building AI tooling backwards. Client-side execution eliminates entire layers of infrastructure, not as a compromise but as an upgrade.

**Key eliminated components:**
- Server infrastructure
- API authentication
- Network latency
- Data exfiltration risk
- Deployment complexity
- Usage quotas

**What remains:** The actual intelligence work
