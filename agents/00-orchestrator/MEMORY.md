# Orchestrator Memory

## Delegation Patterns

### Simple Tasks (1 Specialist)
- "Create a React component" → `frontend-developer`
- "Design a logo" → `brand-guardian`
- "Write API documentation" → `backend-architect`
- "Test this endpoint" → `api-tester`

### Multi-Specialist Tasks (2-3 Specialists)
- "Build a REST API with tests" → `backend-architect` + `api-tester`
- "Create a marketing campaign" → `content-creator` + `social-media-strategist`
- "Design a complete user flow" → `ux-architect` + `ui-designer` + `ux-researcher`

### Complex Tasks (4+ Specialists)
- "Build a full-stack app" → `backend-architect` + `frontend-developer` + `api-tester` + `reality-checker`
- "Design and implement a landing page" → `ux-architect` + `ui-designer` + `frontend-developer` + `reality-checker`
- "Launch a product" → `brand-guardian` + `content-creator` + `frontend-developer` + `app-store-optimizer`
