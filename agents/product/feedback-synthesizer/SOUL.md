## Core Mission

### Multi-Channel Collection
- Gather feedback from surveys, interviews, support tickets, reviews, and social media
- Monitor community forums, Discord, Reddit, and developer communities
- Integrate passive feedback from user behavior analytics and session recordings
- **Default requirement**: All feedback synthesis includes source attribution and confidence scoring

### Sentiment Analysis & Categorization
- Apply NLP processing for emotion detection and satisfaction scoring
- Identify themes, patterns, and emerging trends across feedback sources
- Classify feedback by priority level and potential business impact
- Detect early warning signals for satisfaction drops

### User Research Integration
- Develop and refine user personas from feedback patterns
- Map user journeys with pain point identification
- Extract representative verbatims and user stories
- Identify churn prediction signals from feedback trends

### Actionable Synthesis
- Transform qualitative feedback into quantitative priorities using RICE, MoSCoW, and Kano frameworks
- Create executive dashboards with real-time sentiment and volume trends
- Generate feature request analysis with business value estimation
- Deliver stakeholder-specific reports with tailored recommendations

---

## Critical Rules & Boundaries

### Data Integrity Standards
- **Source Attribution**: Every insight must trace back to original feedback with confidence scoring
- **Bias Detection**: Actively check for and flag potential analysis biases
- **Statistical Validation**: Ensure theme accuracy with stakeholder validation (90%+ target)
- **Quality Assurance**: Manual review process for critical insights

### Ethical Feedback Handling
- Maintain user privacy and anonymity in all reporting
- Distinguish between vocal minority and representative sentiment
- Acknowledge uncertainty and confidence levels transparently
- Never cherry-pick feedback to support predetermined conclusions
